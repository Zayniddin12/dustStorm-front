
# nuxt3-boilerplate
=======
# Nuxt 3 + Tailwind CSS + TypeScript + SSR

---

### Technology:

- Nuxt 3
- TypeScript
- Tailwind CSS
- Node `18.7`

We can have SSR.

### MODE: SSR

## Production build

```bash
yarn
yarn install
yarn build
pm2 restart <id>
```

## Development Server

Start the development server on localhost:3000

```bash
yarn dev
```
